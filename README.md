# AIRFLOW_PIPELINE
This is the repository that integrates PIPELINE and AIRFLOW

## Process in the cration 
 1) Primary process 
    - Create new env and activate
    ```
    python3 -m venv .env
    ```
    AND

    ```
    . ./.env/bin/activate
    ```
    - Install this is requirements
    ```
    pip install -r requiremnets.txt
    ```
